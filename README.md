# beep
🚨 No BS, beautiful status page and uptime monitor. No core model, no SaaS, no fees. 100% open-source, self-hosted, and run locally.

<br>

# Why ?

- Clean and simple UI

- Supports TCP, HTTPS, HTTP, SMTP, ICMP, UDP and DNS

- It just works

- Incident management

- Alerting supported: Email, RSS, and REST API.
